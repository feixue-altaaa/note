

# 按规模分

- 百亿私募：九坤、幻方、灵均、天演、衍复、启林、佳期等
- 低于百亿私募：量派（二十亿）、顽岩（5个亿）、坤望等

# 按资金分

- 资管：基本百亿私募都会做资管，资管就是：公司出策略，投资人出钱买基金（起售额在千万级别），公司会收取管理费，如果赚钱了，公司也会收一部分提成。做资管会有业绩压力。
- 自营：赚的钱都是自己的，没有业绩压力。（幻方的自营规模很大）

# 按交易频率分

- 高频：微妙级延迟，策略不会非常复杂（要考虑延迟），资金容量比较小
- 中低频：开发是二等公民，策略更重要
